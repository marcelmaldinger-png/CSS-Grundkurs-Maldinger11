Was ist CSS?

CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die verwendet wird, um das Aussehen und das Layout von HTML-Dokumenten zu beschreiben.

  • CSS ermöglicht es, das Layout von Webseiten zu gestalten.
  • CSS kann verwendet werden, um Schriftarten, Farben und Abstände zu ändern.
  • CSS kann in HTML-Dokumente eingebettet oder als externe Datei verlinkt werden.
